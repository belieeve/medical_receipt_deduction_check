# Medical Receipt Deduction Checker

医療費控除の対象金額を自動計算・チェックするWebアプリケーションです。
領収証の画像をアップロードすると、OCR（文字認識）機能により金額を自動で読み取り、控除基準額（10万円）を超えているかどうかを瞬時に判定します。

## 機能

- **自動OCR読み取り**: ブラウザ上で領収証画像を解析し、合計金額を抽出します（サーバーへの送信なし）。
- **ドラッグ＆ドロップ**: 複数の画像を一度にアップロード可能。
- **リアルタイム計算**: 合計金額と控除見込み額を即座に計算・表示。
- **プライバシー保護**: すべての処理はブラウザ内で完結するため、個人情報が外部に漏れることはありません。

## 使い方

### 開発環境での実行

1. 依存関係のインストール
   ```bash
   npm install
   ```

2. 開発サーバーの起動
   ```bash
   npm run dev
   ```
   ブラウザで `http://localhost:5173` を開いてください。

### 静的サイト（HTML）としての利用

本アプリは静的なHTMLサイトとして書き出すことができます。

1. ビルドの実行
   ```bash
   npm run build
   ```

2. 生成されたファイルの確認
   `dist` フォルダ内に `index.html` とアセットファイルが生成されます。
   これをWebサーバー（Netlify, Vercel, あるいはローカルサーバー）に配置するだけでサイトとして機能します。

   ※ ローカルで確認する場合は `npm run preview` を推奨します。
